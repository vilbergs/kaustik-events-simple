jQuery with Datatables Example

In this example I create an array from the csv file in php and send it out as a json object, I use the ajax method that ships with datatables to extract the data from the php file and present it.

Datatables provides a method to display a modal when the viewport is too small to display all the data, I decided to make use of that.
